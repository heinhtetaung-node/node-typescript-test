# Frontend Assignment 2023

###  Run project

1. install all dependencies
```
yarn
```

2. run web and API Gateway in parallel
```
npx lerna run dev --parallel
```

3. run test for web and API Gateway in parallel
```
npx lerna run test --parallel
```
